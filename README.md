Tutoriel lancement du projet:
-installer Node.js et ethers.js
-décompresse le zip
-ouvrir 2 terminaux, un dans fullstack/frontend et un dans fullstack/backend
-lancer la commande "nodemon" dans fullstack/backend
-lancer la commande "npm install" puis "npm start" dans fullstack/frontend
-dans un navigateur, aller à l'adresse "http://localhost:4200"
